# Dijkstra Shortest Path Finder (Laravel Project)

This is a simple Laravel project that implements the **Dijkstra Algorithm** to find the shortest path between two nodes (A–I).

## Features

- User Authentication (Register / Login / Logout)
- Dijkstra shortest path algorithm
- Input validation (A–I / a–i support)
- Bootstrap UI design
- Fast path calculation
- Error handling with validation messages

## Tech Stack

- Laravel (PHP Framework)
- Bootstrap 5
- MySQL (optional for auth)
- Blade Templates
